## 246. Strobogrammatic Number


```java

class Solution {
    public boolean isStrobogrammatic(String num) {
        int start = 0, end = num.length() - 1;
        
        while (start <= end) {
            if (!"00 11 88 69 96".contains(num.charAt(start++) + "" + num.charAt(end--)))
                return false;           
        }
        return true;        
    }
}

```

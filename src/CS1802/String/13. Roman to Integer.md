## 13. Roman to Integer ##


```java

class Solution {
    public int romanToInt(String s) {
        int sum = 0;
        Map<Character, Integer> map = new HashMap<>();
        map.put('I', 1);
        map.put('V', 5);
        map.put('X', 10);
        map.put('L', 50);
        map.put('C', 100);
        map.put('D', 500);
        map.put('M', 1000);
        
        int i = 0, j = 1;        
        for (; j < s.length(); i++, j++){
            if(map.get(s.charAt(i)) >= map.get(s.charAt(j)))
                sum += map.get(s.charAt(i));
            else
                sum -= map.get(s.charAt(i));            
        }
        sum += map.get(s.charAt(i));
        return sum;
    }
}


````

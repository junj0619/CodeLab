## 390. Elimination Game

Example:
<pre>
Input:
n = 9,
1 2 3 4 5 6 7 8 9
2 4 6 8
2 6
6

Output:
6
</pre>

```java
class Solution {
    public int lastRemaining(int n) {
        int head = 1, step = 1;
        boolean fromLeft = true;
        
        while (n > 1) {
            if (fromLeft || n % 2 == 1) {
                head += step;                
            }
            step *= 2;
            n /= 2;
            fromLeft = !fromLeft;
        }
        
        return head;
    }
}
```

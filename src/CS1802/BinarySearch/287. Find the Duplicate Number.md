## 287. Find the Duplicate Number

### O(nlogn) solution can be optimzed in O(n)

```java

class Solution {
    public int findDuplicate(int[] nums) {
        //nlogn
        int start = 0, end = nums.length;
        
        while (start < end) {
            int mid = (end - start) / 2 + start;
            int count = 0;
            for (int num : nums) {
                if (num <= mid) count++;
            }
            
            if (count <= mid) {
                start = mid + 1;
            } else {
                end = mid;
            }
        }
        
        return end;
    }
}

```

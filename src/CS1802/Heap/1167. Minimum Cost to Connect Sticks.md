## 1167. Minimum Cost to Connect Sticks

```java
class Solution {
    public int connectSticks(int[] sticks) {
        int cost = 0;
        PriorityQueue<Integer> minHeap = new PriorityQueue<>();
        
        for (int stick : sticks) {
            minHeap.offer(stick);
        }
        
        while (minHeap.size() > 1) {
            int sum = minHeap.poll() + minHeap.poll();
            cost += sum;
            minHeap.offer(sum);
        }
        
        return cost;
    }
}
```

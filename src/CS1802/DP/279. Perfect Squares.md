## 279. Perfect Squares


### Top-Down

```java

class Solution {
    private int[] cache;
    public int numSquares(int n) {
        cache = new int[n];        
        return helper(n);
    }
    
    private int helper(int n) {
        if (n == 0) {
            return 0;
        }
        
        if (cache[n - 1] > 0) {
            return cache[n - 1];
        }
        
        int min = Integer.MAX_VALUE;
        int i = 1;
        while (i * i <= n) {
            min = Math.min(min, helper(n - i * i) + 1);
            i++;
        }
        
        cache[n - 1] = min;
        return min;
    }
}

```


### Bottom-up
```java

class Solution {    
    public int numSquares(int n) {
        int[] cache = new int[n + 1];
        Arrays.fill(cache, Integer.MAX_VALUE);
        
        cache[0] = 0;
        
        for (int i = 0; i <= n; i++) {
            int j = 1;
            while (i + j * j <= n) {
                cache[i + j * j] = Math.min(cache[i + j * j], cache[i] + 1);
                j++;
            }
        }
        
        return cache[n];
    }
}
```

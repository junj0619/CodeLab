## 279. Perfect Squares


### Top-Down

```java

class Solution {
    private int[] cache;
    public int numSquares(int n) {
        cache = new int[n];        
        return helper(n);
    }
    
    private int helper(int n) {
        if (n == 0) {
            return 0;
        }
        
        if (cache[n - 1] > 0) {
            return cache[n - 1];
        }
        
        int min = Integer.MAX_VALUE;
        int i = 1;
        while (i * i <= n) {
            min = Math.min(min, helper(n - i * i) + 1);
            i++;
        }
        
        cache[n - 1] = min;
        return min;
    }
}

```

## 2. Min Moves to Obtain String Without 3 Identical Consecutive Letters

Time complexity: O(n)
Space complexity: O(1)

```java
public int solution(String s) {
    int moves = 0;
    for (int i = 0 ; i < s.length(); i++) {
        int runLength = 1;
        for (; i + 1 < s.length() && s.charAt(i) == s.charAt(i + 1); i++) {
            runLength++;
        }
        moves += runLength / 3;
    }
    return moves;
}
```

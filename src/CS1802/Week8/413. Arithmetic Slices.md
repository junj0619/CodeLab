## 413. Arithmetic Slices ##

### Brute Force O(N^3) ###
```java

class Solution {
    public int numberOfArithmeticSlices(int[] A) {
        //[1,2,3,8,9,10]
        
        int len = A.length;
        int res = 0;
        
        for (int size = 3; size <= len; size++) {
            for (int start = size - 1; start < len; start++) {
                int i = start;
                while (i >= start - (size - 3)) {
                    if ((A[i] - A[i - 1]) != (A[i - 1] - A[i - 2])) break;
                    if (i-- == start - (size - 3)) res++;                                        
                }
               
            }
        }                    
        return res;
    }
}

```

## 259. 3Sum Smaller


```java

class Solution {
    public int threeSumSmaller(int[] nums, int target) {
        int sum = 0;
        int len = nums.length;
        Arrays.sort(nums);
        
        for (int i = 0; i < len - 2; i++) {
            int low = i + 1;
            int hi = len - 1;
            while (low < hi) {
                if (nums[i] + nums[low] + nums[hi] < target) {
                    sum += hi - low;
                    low++;
                } else {
                    hi--;
                }
            }
        }
        
        return sum;
    }
}
```

## 31. Next Permutation

```java

class Solution {
    public void nextPermutation(int[] nums) {
        if (nums == null || nums.length == 0) return;
        
        int len = nums.length;
        int index = len - 2;
        
        while (index >= 0 && nums[index] >= nums[index+1]) {            
            index--;
        }
        
        int end = len - 1;
        if (index >= 0) {            
            while (nums[end] <= nums[index]) end--;
            swap(nums, end, index);            
        }
        
        
        end = len - 1;  
        index++;
        while (index < end) {
            swap(nums, index++, end--);
        }
                
    }
    
    private void swap(int[] nums, int i, int j) {
        int temp = nums[i];
        nums[i] = nums[j];
        nums[j] = temp;
    }
}

```

## 1.Two Sum ##
### Time Complexity: O(N), Space Complexity: O(N) ###
```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        int[] res = new int[2];
        HashMap<Integer, Integer> map = new HashMap<>(); //Key: num, Value: Index
        
        for (int i = 0; i < nums.length; i++) {
            Integer val = map.get(target - nums[i]);
            if (val != null) {
                res[0] = val;
                res[1] = i;
                break;
            }
            map.put(nums[i], i);
        }
        return res;
    }
}
```

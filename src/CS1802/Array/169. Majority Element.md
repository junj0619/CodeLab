## 169. Majority Element

### Boyerâ€“Moore majority vote algorithm

```java

class Solution {
    public int majorityElement(int[] nums) {
        int count = 0, max = 0;
        for (int num : nums) {
            if (count == 0) {
                max = num;
                count++;
            } else if (num == max) 
                count++;
            else 
                count--;            
        }
        return max;
    }
}
```
